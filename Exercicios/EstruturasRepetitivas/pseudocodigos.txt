Construir um algoritmo que calcule a média aritmética de vários valores inteiros positivos, lidos externamente. 
O final da leitura acontecerá quando for lido um valor negativo.

Escrever um algoritmo que leia um número de valores, calcule a média aritmética dos valores lidos, 
a quantidade de valores positivos, a quantidade de valores negativos. Mostre os resultados. 

Escrever um algoritmo que lê 10 valores, um de cada vez, e conta quantos deles estão no intervalo [10,20] 
e quantos deles estão fora do intervalo, escrevendo estas informações. 

Faça um algoritmo estruturado que leia uma quantidade não determinada de números positivos.
Calcule a quantidade de números pares e ímpares, a média de valores pares e a média geral dos
números lidos. O número que encerrará a leitura será zero.

Escreva um algoritmo que leia uma sequência de números do usuário e realize a
soma desses números. Encerre a execução quando um número negativo for digitado.

Algoritmo1
Var 
    media, numero, contador, soma: inteiros
Inicio

    Escreva("Digite um numero: ")
    Leia(numero)
    contador <- 0
    Enquanto numero > 0:
        soma = soma + numero
        contador = contador + 1
        Escreva("Digite um numero: ")
        Leia(numero)
    FimEnquanto

    media = soma / contador
    Escreva("A media é:", media)

Fim

Algoritmo2
Var 
    media, qtd, contador, qtdPositivos, qtdNegativos, numero: inteiros
Inicio

    Escreva("Quantos valores vocês deseja calcular a media? ")
    Leia(qtd)
    Enquanto qtd < 0:
        Escreva("Quantidade inválida, digite novamente:")
        Leia(qtd)
    FimEnquanto

    contador <- 0
    qtdNegativos <- 0
    qtdPositivos <- 0
    Enquanto contador =! qtd:
        Escreva("Digite um número: ")
        Leia(numero)
        Se numero > 0:
            qtdPositivos = qtdPositivos + 1
        Senão:
            qtdNegativos = qtdNegativos + 1
        FimSe
        soma = soma + numero
        contador = contador + 1
    FimEnquanto

    media <- soma / qtd

    Escreva("Media", media, "qtdNegativos", qtdNegativos, "qtdPositivos", qtdPositivos)
Fim
